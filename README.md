# Hướng dẫn cài đặt
## 1. Server lgo
Yêu cầu:
- Maven 3.8.6
- JDK java 1.8
Script chạy dự án lgo: 
**mvn spring-boot:run**
## 2. Giao diện app
Yêu cầu: 
- Nodejs v16
Script chạy dự án app:
**npm install**
**npm start**
## 3. Tài liệu sử dụng
Yêu cầu:
- Nodejs v16
- Yarn lasted version
Script chạy dự án app:
**yarn cache clean**
**yarn install**
**yarn start --port 3001**
